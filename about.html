Our Coding Club

What is Coding Club?

Coding Club like its name suggests is a school club that focuses its learning and development on digital skills, for example students can pick two different coding types being either normal coding or robotics, students than use skills they've learned from coding club into upcoming events to win prizes.

What do we learn from Coding Club?

Coding Club teaches different year group different skills, for example juniors would learn coding languages such as block coding which involve coding on basic websites like scratch and code.org. Seniors on the other hand would learn text based coding which is more complex and used on apps like Javascript and python, this is important because we are teaching both groups important skills to prepare them for school internals or upcoming competitions involving the Karamu Coding Club

Who can help you in Coding Club

If you're struggling while coding, we have teachers and year 13's who have experience with coding and are happy to help.
